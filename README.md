# Rvmat-Creator

一个用于快速处理 DayZ 材质 Rvmat 文件的软件。

## 功能

- 快速生成损坏材质文件
- 简单易用的图形界面
- 自动处理 .rvmat 文件格式

## 使用方法

1. 运行程序:
   ```
   python src/main.py
   ```

2. 在界面中选择要处理的 .rvmat 文件

3. 点击"生成损坏材质文件"按钮

4. 程序将生成三个新文件:
   - 原文件名_worn.rvmat
   - 原文件名_damage.rvmat
   - 原文件名_destruct.rvmat

## 开发

该项目使用 Python 编写，无需额外依赖，仅使用标准库。
